# Function: getItemOnce()

> **getItemOnce**(`key`): `string`

根据key值获取cookie数据后删除Cookie中该键值对

## Parameters

• **key**: `string`

key值

## Returns

`string`

Cookie中键值为key的值

## Example

```ts
getItemOnce('test');
```
